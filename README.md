# goit-algo2-hw-04

## GOIT Design and Analysis of Algorithms Homework 4

### Requirements

- Python 3.10+
- Git

### Usage

Clone this repository:
```shell
git clone https://github.com/AntonChubarov/goit-algo2-hw-04.git
```

Navigate to project directory:
```shell
cd goit-algo2-hw-04
```

Run unit tests:
```shell
python3 -m unittest
```
